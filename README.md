# Prism
Design of the Prism16 retro computer.

This project is still in the design phase, and goals may vary wildly depending on how things progress.

The current working idea is to have a computer that will have some of the same power of a SuperNES and an AppleII GS, but with a few extras to make it unique.

At present the target CPU will be a 65C816 running at 16MHz main clock.

Graphics will be done through an FPGA with a target resolution of 320x240 with some of the same hardware features found in the SNES.

Audio at present will be driven by a Yamaha OPL-3 FM synth chip, but may visit the idea of adding some hardware mixed PCM audio; similar to what is found on the Amiga Paula chip.
